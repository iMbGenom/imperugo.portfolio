Galaxy

A Gorgeous & Multi-page Template For Portfolio

  • Version: 1.0.0
  • Author: LineThemes
Thank you for purchasing my theme. If you have any questions that are beyond the scope of this documentation, please feel free to email us via the contact form here!

Please be noticed that all the parameters used to install "Contact Form", "Auto Responder", "Mailchimp Integration" are placed in the file "php/config.php"

HTML Structure

A basic HTML page in Galaxy has the following structure: 
<header id="line-header">
	<!-- Page header content -->
</header>

<div id="line-content-top">
	<!-- Content of popup navigator -->
</div>

<div id="line-page-body">
	<div class="line-container">
		<div class="line-page-content">
			<!-- Page content -->
		</div>
	</div>
</div>

<div id="line-content-bottom">
	<!-- Content bottom area -->
</div>

<footer id="line-footer">
	<!-- Footer content -->
</footer>

CSS Structure

Galaxy template was basically built on LESS. All the LESS files are located in the folder assets/less and could be edited with some popular editors, such as NotePad++, Sublime Text,ect.

Here is the list of LESS files used in this template:

  • assets/less/config.less (Template variables)
  • assets/less/responsive.less (Responsive styles)
  • assets/less/shortcode.less (Shortcode styles)
  • assets/less/woocommerce.less (WooCommerce styles)
  • style.less (Template styles)

Style Customization

Variables

We provide variables to help you with the template customization. (You can modify the value of variables in the file "assets/less/config.less")

  • @layout-width: Width display content settings.
  • @body-background: Background color and image settings.
  • @submenu-width: Width submenu settings.
  • @scheme: Main color scheme of template.
  • @body-font: Fontface applied to the whole website (body).
  • @heading-font: Fontface applied exclusively to heading H1, H2,..., and H6.

Sidebar Position

Galaxy template provides single sidebar including 2 classes by default to customize the display content.
  • line-sidebar-left: To move the sidebar to the left of the content.
  • line-sidebar-right: To move the sidebar to the right of the content.

Footer Layout

In Galaxy template, footer part is arranged as a grid, divided into 12 columns. Each component in the footer will be determined the width spread from 1 to 12 columns, according to the releven class as line1, line2, ..., line12.

For example:

<div id="line-content-bottom">
	<div class="line-wrap line-sidebar-bottom">
		<!-- Element 1 -->
		<div class="line-column line3">
			<aside class="widget widget_text">
				<div class="textwidget">
					<!-- Widget content -->
				</div>
			</aside>
		</div>

		<!-- Element 2 -->
		<div class="line-column line3">
			<aside class="widget widget_text">
				<div class="textwidget">
					<!-- Widget content -->
				</div>
			</aside>
		</div>
	</div>
</div>

Portfolio Layout

Galaxy used the grid style to display the portfolio items. Each row can present maximum 4 portfolio items.

In order to change the number of portfolio displaying in the same row, you could insert the class "line-col-x", (with x is the amount of displaying column) for the div tag with the CLASS as "line-content"

For instance,

<div class="line-content line-content-isotope line-col-4">

Portfolio Styles

With the purpose of serving the demand of designing portfolio, Galaxy has already been designed with 2 default styles for portfolio. Each of them will be correlative with one of the below:

  • line-portfolio-bg (Demo)
  • line-portfolio-ajax (Demo)

In order to establish the portfolio style, you just need to insert one of these two classes to put in the DIV tag with the CLASS as line-portfolio

For example:

<div class="line-wrap line-portfolio line-portfolio-bg">
    ...
</div>

MailChimp Integration

  • mailchimp_key: MailChimp provides the API Key. You can take the API Key from this instruction: http://kb.mailchimp.com/article/where-can-i-find-my-api-key 
    $config['mailchimp_key'] = '13c3e4b3ea4c85d2ca33b4be3b75de2c-us3';
  • mailchimp_list_id: List ID was created in MailChimp.com. You can see the instruction to take the “List ID” here: http://kb.mailchimp.com/article/how-can-i-find-my-list-id 
    $config['mailchimp_list_id'] = '3e329a10f8';
  • mailchimp_success_message: This is the message to inform users after successfully subscribed. 
     $config['mailchimp_success_message'] = 'Your success message';

AJAX Contact Form

Currently, there are 3 parameters to establish the “Contact form”

  • email_to: Customer's feedback recipient. 
    $config['email_to'] = 'support@linethemes.com';
  • default_subject: The feedback email subject will be sent to the above established email address. 
    $config['default_subject'] = 'New contact message from linethemes.com';
  • success_message: The notification to customer when their email is successfully sent. 
    $config['success_message'] = 'Thank you! Your message has been sent to us';

You can change the email content by editing the file "php/tmpl/mail_content.php"

Auto Respond

  • To activate the “Auto Respond” function, you need to modify the value of the parameter $config['auto_response'] to true. 
    $config['auto_response'] = true;
  • auto_response_name: This is the sender's name. 
    $config['auto_response_name'] = 'Linethemes Support';
  • auto_response_email: This is sender's email address. This parameter will be used to receive the reply email from the user if necessary. 
    $config['auto_response_email'] = 'support@linethemes.com';
  • auto_response_subject: This is the email's subject that will be sent to the user. 
    $config['auto_response_subject'] = 'Thank you for contacted me from linethemes.com';
